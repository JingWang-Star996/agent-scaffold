# 自验证机制（Self-Verification）

输出必须验证，不信任自己的"我觉得对了"。

## 验证清单

- **文件操作**：写完文件后 read 回来确认内容正确
- **命令执行**：检查 exit code + 实际输出，不信 "应该没问题"
- **数据处理**：用脚本处理的数据，抽样验证结果
- **子代理输出**：要求返回可验证的 handle（URL/路径/状态码），不轻信自述
- **配置变更**：变更后立即验证生效

## 何时加载此 Skill

- 当你完成文件操作后
- 当你执行了重要命令后
- 当你处理了批量数据后
- 当你依赖子代理的输出时
- 当你修改了配置后

## 反模式（禁止）

- ❌ "应该没问题" — 必须验证
- ❌ "我觉得对了" — 必须检查
- ❌ 子代理说"完成"就信了 — 必须要求可验证的 handle
